# 🚀 Rocket Telegram Reporter

Массовая отправка жалоб в Telegram на пользователей, чаты, каналы и ботов.

## 📌 Возможности

- Жалобы на **пользователей, чаты, каналы, ботов**
- **10 причин** для жалоб (спам, насилие, порнография, мошенничество и др.)
- **Мульти-аккаунты** (до 4+ номеров)
- **Автоматическая авторизация** с сохранением сессий
- **Задержки между жалобами** для избежания блокировок
- **Работа на VPS** в фоновом режиме
